# Password Generator 🔐

A simple program that helps you create strong and random passwords quickly. It includes everything you need: uppercase and lowercase letters, numbers, and special symbols.

---

## Features ✨

- Generate strong passwords.  
- Set the password length (from 8 to 50 characters).  
- Generate up to 50 passwords at once.  
- Easy and user-friendly interface.  

---
